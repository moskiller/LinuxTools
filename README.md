# LinuxTools
用于VPS上的一些脚本合集    

为了方便自己操作VPS，借用一下各位大佬的脚本。。。


## 下载

```shell
wget -N https://raw.githubusercontent.com/moskiller/LinuxTools/main/linuxtools.sh && chmod -R 777 linuxtools.sh && bash linuxtools.sh
```


## 运行
```shell
bash linuxtools.sh
```
