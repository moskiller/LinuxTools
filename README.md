# LinuxTools
用于VPS上的一些脚本合集    

为了方便自己操作VPS，借用一下各位大佬的脚本。。。

## 使用说明：

1.脚本需要root权限运行

2.日志文件保存在 /var/log/linuxtools.log

3.所有重要操作都会要求确认

4.安装过程会显示进度指示器

5.出错时会记录到日志并显示错误信息


## 下载

```shell
wget -N https://raw.githubusercontent.com/moskiller/LinuxTools/main/linuxtools.sh && chmod -R 777 linuxtools.sh && bash linuxtools.sh
```


## 运行
```shell
bash linuxtools.sh
```
